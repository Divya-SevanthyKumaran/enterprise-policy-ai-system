# Enterprise Policy AI System

## 📌 Overview
An AI-powered web application built using Django that allows users to query enterprise policy documents using natural language. The system uses Retrieval-Augmented Generation (RAG) for document-based queries and LLM for general questions.

---

## 🚀 Features
- Natural language question answering
- Document upload and processing
- Semantic search using vector embeddings
- RAG-based document responses
- LLM-based general responses
- Chat interface with history clear option
- User authentication system

---

## 🛠️ Tech Stack
- Backend: Django (Python)
- Database: MySQL
- AI: Sentence Transformers, RAG, LLM
- Vector Store: Pinecone / Local DB
- Frontend: HTML, CSS

---

## ⚙️ How It Works
1. Upload policy documents
2. Convert documents into embeddings
3. Store embeddings in vector database
4. User asks question
5. System:
   - Uses RAG for document queries
   - Uses LLM for general queries
6. Displays response in chat interface
