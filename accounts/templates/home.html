What This Project Does

Sahi Enterprise is an AI-powered document search platform built using Retrieval-Augmented Generation (RAG).

Users can upload documents, and the system converts them into embeddings stored in a vector database. When a question is asked, the system retrieves the most relevant content and generates accurate answers using AI models.

This enables fast, intelligent, and context-aware information retrieval from large datasets.
